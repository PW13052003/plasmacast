What This Means

MAE (Mean Absolute Error) measures how many donors off our predictions are on average per day. We show it as a percentage of each city's average donor count to make fair comparisons across cities of very different sizes. An MAE of 15 donors means very different things for NYC (318 avg) vs Jacksonville (38 avg) — so percentage normalizes for that difference. The tight 5.4%–6.9% range across all 10 cities shows the model generalizes well regardless of city size.

About PlasmaCast

PlasmaCast is an end-to-end machine learning system that forecasts daily plasma donor demand across 10 major US cities. Built as a portfolio project inspired by the CSL Behring real-world data challenge.
